# Django Swagger

Auto generate documentation about django rest framework features and quickly generate **Schema Views** for a given base url endpoint.

# Requirements and How to Use

1. Django Rest Framework version 3.5 or higher
2. Django Swagger `pip install django-rest-swagger`
3. Virtual Environment
4. Clone or download this repositories
5. Added `rest_framework_swagger` in your `INSTALLED_APPS`
6. Runserver with `py manage.py runserver`
